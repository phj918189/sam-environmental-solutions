## 1. 요약

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 삼양건설환경연구소 공식 웹사이트 |

## 2. 프로젝트 개요

### 2-1. 왜 이 웹사이트를 만들었나요?

| 목적 | 설명 |
|------|------|
| **회사 정보 제공** | 1992년 설립, 30년 이상 경험, 7종 인증 등 회사 정보를 한곳에 모아 전달 |
| **사업 영역 소개** | 대기·수질·악취 환경측정, 설계·시공 등 서비스를 명확히 소개 |
| **견적 문의 유도** | 방문자가 전화·이메일·문의 폼을 통해 쉽게 연락할 수 있도록 구성 |
| **사업실적 showcase** | 운영 시설, 시공 사업, 휴게소 등 실적을 통해 신뢰 강화 |

### 2-2. 웹사이트가 제공하는 가치

| 가치 | 내용 |
|------|------|
| **24시간 정보 제공** | 사업시간 외에도 회사 소개·연락처 확인 가능 |

| **검색 노출** | 검색엔진 최적화로 “대전 환경측정”, “수질분석” 등 검색 시 노출 |
| **첫인상** | 단순 명함·전화 대비, 웹으로 보다 전문적·신뢰감 있는 이미지 제공 |

## 4. 주요 기능

### 4-1. 견적 문의 폼

| 항목 | 설명 |
|------|------|
| **기능** | 방문자가 성함, 회사, 연락처, 문의 유형, 내용을 입력 후 전송 |
| **동작 방식** | Web3Forms 서비스를 통해 **지정한 이메일로 자동 발송** |
| **비용** | Web3Forms 무료 플랜 사용 (무제한) |

| 항목 | 내용 |
|------|------|
| **표시 정보** | 주소, 전화번호, 이메일, 영업시간, 휴무일 |
| **지도** | Google Maps 임베드, 위치 확인 및 새 탭에서 길찾기 |
| **수정 방법** | `src/data/contact.ts` 파일 한 곳만 수정하면 전체 사이트에 반영 |

| 항목 | 설명 |
|------|------|
| **전환 방식** | 우측 상단 언어 버튼 클릭 |
| **URL 규칙** | 한국어: `/about`, 영어: `/en/about` 등 |
| **번역 관리** | `src/locales/ko.ts`, `en.ts` 파일에서 일괄 관리 |

## 6. 검색 노출(SEO)

### 6-1. SEO란?

**SEO(Search Engine Optimization)**는 구글 등 검색엔진에서 “대기환경 측정”, “대전 수질분석” 등을 검색했을 때 우리 사이트가 상위에 노출되도록 맞춰두는 작업입니다.

### 6-2. 적용된 SEO 항목

| 항목 | 설명 |
|------|------|
| **메타 태그** | 페이지 제목, 설명, 키워드가 검색 결과에 반영되도록 설정 |
| **Open Graph** | SNS 공유 시 미리보기 제목·설명·썸네일 표시 |
| **사이트맵** | 검색엔진이 모든 페이지를 쉽게 찾아가도록 경로 제공 |
| **robots.txt** | 검색엔진이 수집해도 되는 페이지·제외할 페이지 안내 |
| **구조화 데이터** | 회사명·주소·전화번호 등 정보를 검색엔진이 읽기 쉬운 형식으로 제공 |

### 6-3. 도메인 변경 시

| 상황 | 처리 |
|------|------|
| **사이트 주소 변경** | `.env`에 `VITE_SITE_URL=새주소` 설정 후 빌드 |
| **자동 반영 대상** | 메타 태그, 사이트맵, robots.txt 등 SEO 관련 항목 |

## 7. 배포 및 도메인

### 7-1. 배포 플랫폼 권장

| 플랫폼 | 설명 |
|--------|------|
| **Vercel** (권장) | 현재 프로젝트 구조에 맞춰 설정되어 있어 추가 수정 없이 배포 가능 |
| **대안** | Netlify, Cloudflare Pages, GitHub Pages 등 사용 가능 |

### 7-2. 커스텀 도메인 사용 절차

| 단계 | 할 일 |
|:----:|-------|
| 1 | 사용할 도메인 확보 (예: 가비아, GoDaddy) |
| 2 | Vercel에 프로젝트 배포 후, Domains 메뉴에서 해당 도메인 추가 |
| 3 | 도메인 업체 관리 페이지에서 Vercel이 안내하는 DNS 레코드 설정 |
| 4 | 24시간 이내 도메인 연결 완료 후 접속 확인 |

### 7-3. 도메인 추천 예시

| 번호 | 도메인 예시 | 비고 |
|:----:|-------------|------|
| 1 | samyang-environment.com | 직관적 |
| 2 | samyangenv.com | 짧고 기억하기 쉬움 |
| 3 | samyang-research.com | 연구소 이미지 강조 |
| 4 | samyang-env-lab.com | 환경·연구소 조합 |
| 5 | samyangenvironmental.com | 영문 회사명 |

※ 실제 사용 가능 여부는 도메인 등록 업체에서 조회해야 합니다.

## 8. 유지보수 안내

| 구분 | 수정 가능 내용 | 필요 지식 |
|------|-----------------|-----------|
| **일반 담당자** | 연락처, 영업시간 등 기본 정보 | `src/data/contact.ts` 안내만 숙지 |
| **운영 담당자** | 뉴스·자료 등록, 사업실적 추가 | `src/data/` 폴더 내 관련 파일 |
| **개발자** | 레이아웃, 디자인, 새 기능 | 프론트엔드 개발 지식 |

### 8-2. 자주 수정되는 항목

| 항목 | 수정 위치 | 영향 범위 |
|------|-----------|-----------|
| 연락처(주소, 전화, 이메일) | `src/data/contact.ts` | 전체 사이트 |
| 사업실적 | `src/data/portfolio.ts` | 사업실적 페이지 |
| 소식·공지 | `src/data/news.ts` | 소식 페이지 |
| 회사 소개 내용 | `src/data/about.ts` | 회사소개 페이지 |

### 8-3. 유지보수 비용 개요

| 항목 | 예상 |
|------|------|
| **호스팅(배포)** | Vercel 무료 플랜 사용 시 $0 |
| **문의 폼** | Web3Forms 무료 플랜 사용 시 $0 |
| **도메인** | 연간 약 1~2만 원 수준 (업체에 따라 상이) |
| **유지보수 인력** | 소규모 수정은 월 수 시간 내 처리 가능 |

| **도메인** | 웹사이트 주소 (예: www.samyang-env.co.kr) |

| **호스팅** | 웹사이트 파일을 저장·제공하는 서버 |
| **SEO** | 검색엔진 최적화 (검색 시 잘 노출되도록 설계) |
| **API** | 다른 서비스와 연동할 때 사용하는 인터페이스 |
| **환경 변수** | 서버·프로그램 설정에 사용하는 비공개 값 (API 키 등) |
